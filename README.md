# HUT HMKG Ke-79 🌤️

Website resmi untuk memperingati **Hari Meteorologi, Klimatologi, dan Geofisika (HMKG) Ke-79** Tahun 2026 — *"BMKG Hebat, Indonesia Berdaulat, Adil dan Makmur."*

🔗 **Live Site:** [infobmkg.github.io/hmkg79](https://infobmkg.github.io/hmkg79/)

---

## 📖 Tentang

Website ini dibuat untuk mendukung rangkaian acara peringatan HUT HMKG Ke-79, menampilkan informasi lomba, galeri kegiatan, jadwal acara, sponsor, hingga kontak panitia.

## ✨ Fitur

- **Beranda** — Hero section dengan hitung mundur menuju hari puncak acara
- **Live Streaming** — Tautan langsung ke siaran YouTube saat acara berlangsung
- **Lomba** — Informasi kompetisi (Fotografi, Poster Digital, Video Kreatif, Karya Tulis, Data Cuaca, dll.)
- **Galeri** — Dokumentasi kegiatan dengan filter kategori (Upacara, Pembukaan, Lomba, Penutupan)
- **Sponsor** — Daftar mitra dan pendukung acara
- **Jadwal & Tahapan** — Rangkaian agenda dan alur pendaftaran peserta
- **Kontak** — Informasi contact person, email, WhatsApp, dan lokasi kegiatan
- **Media Sosial** — Tautan ke Instagram, Twitter/X, YouTube, TikTok, dan Facebook resmi InfoBMKG


## 🛠️ Teknologi

Website statis yang di-hosting menggunakan **GitHub Pages**, dibangun dengan HTML, CSS, dan JavaScript.
